Linear Equations

A linear equatuion is any equation that can be writte

ax + b = 0

where a and b are real numbers and a is a variable. This Form of a linear equation. Note that most linear equations will not start off in this form. Also, the variable may or may not be an x

Quadratic Equation

The standard form of a quadratic looks like this:

ax2 + b = 0

where a, b, a are know values and x is the variable or unknown (we don't know it yet). Also, a can't be 0.

Intersection

The intersection of two sets A and B, denoted by A B, is the set all objects thet are members of both the sets A and B. In symbols,

A = A ∩ B = { x : x ∈ A and x ∈ B }

That is, x is an element of the intersection A B if and only if x is both an element of A and an element of B.
